Aurora Mdl Tools

Adds support for mdl file format to blender 2.6x

  • Latest Release
  • View On GitHub

This project is maintained by gyoerkaa

About

This blender Add-On adds support for the MDL file format. Meshes, textures and animations are fully supported. Support for particle systems is limited: While import and export is possible, they won't be displayed or rendered.

Installation

Aurora MDL Tools have to be installed manually. An Add-On is considered installed, when it is located in the ../scripts/addons folder. Extracting the auroramdltools folder into the scripts/addons directory is enough.

Once the Add-On has been installed, it has to be enabled before it can be used. Start blender and navigate to File and User Preferences. Select Addons andenable Aurora Mdl Tools by ticking the check box (It is located under the Import-Export category). Make sure to click on the Save as Default /Save User Settings Button, if you want Aurora MDL Tools to be enabled by default, else you will have to re-enable it after every startup.
